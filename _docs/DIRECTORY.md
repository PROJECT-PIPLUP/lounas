# ディレクトリ構成

この構成に遵守すること。

## コンポーネント

コンポーネントは `src/components/{グループ}` 以下に作ること。

> **Note**
>
> 例）`Footer`
>
> `src/components/navigations/Footer.tsx`

## コンポーネントのスタイル

コンポーネントのスタイルは `src/styles/components/{グループ}` 以下に作ること。

> **Note**
>
> 例）`loading`
>
> `src/styles/components/layouts/loading.module.scss`
